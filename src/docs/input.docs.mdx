## Usage

### Input with Icon

An Input can include an icon inside the input field.

<Canvas of={stories.withIcon} />

### Input `isDisabled`

When `isDisabled` is set to `true`, the Input will be disabled and cannot be interacted with.

<Canvas of={stories.isDisabled} />

### Input `isInvalid`

When `isInvalid` is set to `true`, the Input will be styled to indicate that the input is invalid.

<Canvas of={stories.isInvalid} />

### Input `isBorderless`

When `isBorderless` is set to `true`, the Input will be rendered without a border.

<Canvas of={stories.isBorderless} />
