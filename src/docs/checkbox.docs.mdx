## CheckboxGroup

A `<CheckboxGroup />` allows a user to select multiple items from a list of
options.

It can be composed with a `<Label />` and `<Description />` to correctly label
the input to assistive technologies, and provide extra context.


<Unstyled>
<div className="grid grid-cols-1 gap-2 md:grid-cols-[2fr_1fr] items-center">
    ```tsx
    import { Checkbox, CheckboxGroup,Description, Label } from "boondoggle";

    <CheckboxGroup>
        <Label>This is the label</Label>
        <Checkbox value="option_1" label="Option 1" />
        <Checkbox value="option_2" label="Option 2" />
        <Checkbox value="option_3" label="Option 3" />
        <Description>This is the description</Description>
    </CheckboxGroup>
    ```
    <Canvas of={checkboxGroupExamples.example} sourceState="none" />
</div>
</Unstyled>

When used in this way, the `<CheckboxGroup />` will manage the state of the
selected items, and provide a way to access the selected values. Each `<Checkbox
/>` should have a `value` prop that is unique to the group, this is what will be
provided via the `onChange` callback.

<ArgTypes of={CheckboxGroup} include={["onChange"]} />
