# CheckboxGroup

[source code](https://github.com/alex-mcgovern/boondoggle/tree/main/src/components/checkbox)
&nbsp;|&nbsp;
[react aria](https://react-spectrum.adobe.com/react-aria/CheckboxGroup.html)

A CheckboxGroup allows a user to select multiple items from a list of options.

It can be composed with a Label and Description to correctly label the input to
assistive technologies, and provide extra context.

<Canvas of={checkboxGroupExamples.example} source="none" />

When used in this way, the CheckboxGroup will manage the state of the selected
items, and provide a way to access the selected values. Each `<Checkbox />`
should have a `value` prop that is unique to the group, this is what will be
provided via the `onChange` callback.

<ArgTypes of={CheckboxGroup} include={["onChange"]} />

### CheckboxGroup `isInvalid`

When `isInvalid` is set to `true`, the group will be styled to indicate that the
user has not selected a valid option.

<Canvas of={checkboxGroupExamples.isInvalid} />

### CheckboxGroup `isDisabled`

When `isDisabled` is set to `true`, the group will be styled to indicate that
the user cannot interact with the options.

<Canvas of={checkboxGroupExamples.isDisabled} />
